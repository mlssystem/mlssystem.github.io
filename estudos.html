Sites que uso como refências para estudos, e pesquisas…

Quando me interessei por tecnologia!

Comecei pesquisar, e confesso que fiquei um pouco perdido, pois essa área não só está em constante desenvolvimento como também tem muitos seguimentos, como por exemplo:

  • Desenvolvimento para Web
  • Desenvolvimento de Software
  • Robótica
  • Redes
  • Hardware
  • Mobile
  • Gerenciamento de Banco de Dados
  • Nuvens
  • Sistemas Operacionais
  • Segurança
  • Etc…

E cada seguimento, diversas ramificações!

Depois de algumas pesquisas! Aprendi que para ser um desenvolvedor tanto para web como para server o ideal é começar estudando Algoritmo, posteriormente o basico para web; HTML, CSS, Java Script, e PHP.

Ja que é para estudar. Vamos conhecer as linguagens de marcação, e de programação tambem!

Pensando assim! Para 'back-end' escolhi PHP para interagir com a web, e Java, pois além de ser multplataforma é uma linguagem sólida, popular e está em constante desenvolvimento.

Acho inetressante ter conhecimento em mais de um Sistema Operacional, e por isso ecolhi Linux, me indentifico com a ideia de que todos possam ter acesso a um Sistema Operacional sem ter que pagar por ele, além de ser código aberto, ter várias distros, também tem uma comunidade 100% ativa, e totalmente disposta a ajudar os iniciante através.

Durante algumas pesquisas, e vídeo aulas, conheci os sites abaixo, e por isso uso como base para estudos e pesquisas.
